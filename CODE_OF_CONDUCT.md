# Code of Conduct

This project aims to be a focused, respectful place to improve an unofficial
Go SDK for the Codex app-server protocol.

## Expected Behavior

- Be kind, direct, and constructive.
- Assume good intent while still reviewing code and claims carefully.
- Keep discussions technical, evidence-based, and relevant to the project.
- Respect maintainers' time by using clear reproduction steps and small,
  reviewable changes.
- Protect user privacy. Do not post credentials, private workspaces, private
  prompts, account data, or proprietary schema dumps.

## Unacceptable Behavior

- Harassment, insults, threats, or discriminatory language.
- Publishing another person's private information.
- Posting secrets, private account data, or exploit details in public threads.
- Repeatedly derailing issues or pull requests after maintainers ask to return
  to scope.
- Misrepresenting this project as official OpenAI software.

## Enforcement

Maintainers may edit, hide, or remove comments; close issues or pull requests;
or temporarily/permanently restrict participation when behavior harms the
project or its contributors.

If you need to report a conduct concern, contact the maintainers through the
repository's normal private channels when available. For security-sensitive
reports, follow `SECURITY.md` instead.
